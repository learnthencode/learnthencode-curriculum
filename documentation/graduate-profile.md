# LearnThenCode Graduate Profile

## Introduction

A LearnThenCode graduate is a learner who has developed the technical foundation, engineering mindset, communication skills, and practical experience required to begin a career in software engineering.

Graduates are not defined only by the technologies they know, but by their ability to understand problems, design solutions, build software, and continue learning.

---

# Technical Skills

A LearnThenCode graduate should be able to:

- Build responsive web applications.
- Understand frontend and backend development concepts.
- Write clean, readable, and maintainable code.
- Use Git and GitHub for version control.
- Work with modern frontend technologies.
- Build backend applications and APIs.
- Design and interact with databases.
- Debug and improve existing software.

---

# Software Engineering Skills

A graduate should demonstrate the ability to:

- Analyze problems before implementation.
- Break complex problems into smaller tasks.
- Read and understand unfamiliar code.
- Test software behavior.
- Identify and fix bugs.
- Follow professional development workflows.
- Apply good software engineering practices.

---

# Communication Skills

A LearnThenCode graduate should be able to:

- Explain technical concepts clearly.
- Present software projects confidently.
- Document their work.
- Collaborate with other developers.
- Participate in technical discussions.

---

# Professional Mindset

Graduates should demonstrate:

- Curiosity.
- Discipline.
- Problem-solving ability.
- Willingness to receive feedback.
- Commitment to continuous improvement.

---

# Project Experience

A graduate should complete:

- Multiple practical coding labs.
- Mini-projects demonstrating specific skills.
- A final capstone project demonstrating full stack development ability.

---

# Final Goal

The goal of LearnThenCode is to produce graduates who are not only capable of writing code, but who understand software engineering as a discipline and can continue growing beyond the bootcamp.
