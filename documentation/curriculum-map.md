# LearnThenCode Curriculum Map

## Overview

The LearnThenCode Full Stack Software Engineering Bootcamp is structured as a progressive learning journey.

Each course builds upon previous knowledge, allowing learners to develop from foundational concepts into full stack software engineering skills.

---

# Course Structure

## Course 1: Developer Foundations

Learners are introduced to:

- Software engineering concepts.
- Development environments.
- Command line fundamentals.
- Git and GitHub.
- Professional developer workflows.

---

## Course 2: Web Development

Learners explore:

- How the web works.
- HTML.
- CSS.
- Responsive design.
- Browser fundamentals.

**Milestone Project:**

Frontend mini-project.

---

## Course 3: Programming with JavaScript

Learners develop programming skills through:

- JavaScript fundamentals.
- Variables and data types.
- Functions.
- Arrays and objects.
- DOM manipulation.
- Events.
- APIs.
- JSON Server.
- CRUD operations.

**Milestone Project:**

JavaScript application.

---

## Course 4: Frontend Engineering

Learners learn:

- React fundamentals.
- Components.
- Props.
- State.
- Hooks.
- Routing.
- Forms.
- API integration.

**Milestone Project:**

React application.

---

## Course 5: Backend Engineering

Learners explore:

- Backend development concepts.
- APIs.
- Server-side programming.
- Authentication.
- Authorization.
- Application architecture.

---

## Course 6: Databases

Learners understand:

- Data modeling.
- SQL fundamentals.
- Database relationships.
- CRUD operations.
- Database integration.

---

## Course 7: Full Stack Engineering

Learners combine previous knowledge to:

- Build complete applications.
- Connect frontend and backend systems.
- Apply professional development practices.
- Deploy applications.

**Final Project:**

Full stack capstone project.

---

# Learning Progression

The curriculum follows this progression:

```
Foundations
     ↓
Web Fundamentals
     ↓
Programming Fundamentals
     ↓
Frontend Development
     ↓
Backend Development
     ↓
Databases
     ↓
Full Stack Applications
```
