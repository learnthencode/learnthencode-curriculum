# LearnThenCode Assessment Framework

## Overview

LearnThenCode assessments are designed to measure understanding, practical ability, problem-solving skills, and professional growth.

Assessment is not only about checking whether learners can write code, but whether they can apply knowledge effectively.

---

# Assessment Methods

## Coding Labs

Labs measure a learner's ability to apply individual concepts.

Examples:

- Completing tasks.
- Debugging code.
- Passing automated tests.
- Implementing features.

---

## Mini Projects

Mini-projects evaluate a learner's ability to combine multiple concepts.

Learners demonstrate:

- Creativity.
- Problem solving.
- Application of concepts.
- Code organization.

---

## Project Presentations

Learners present their projects to develop:

- Communication skills.
- Technical explanation skills.
- Confidence.

---

## Capstone Project

The capstone project represents the culmination of the bootcamp.

Learners demonstrate their ability to:

- Plan software solutions.
- Build full applications.
- Apply frontend and backend knowledge.
- Work with databases.
- Present technical decisions.

---

# Feedback Philosophy

Feedback should help learners improve.

Effective feedback should be:

- Clear.
- Specific.
- Constructive.
- Actionable.

---

# Measuring Growth

Assessment should consider:

- Technical progress.
- Problem-solving ability.
- Communication.
- Professional habits.
- Continuous improvement.

---

# Final Goal

The purpose of assessment is to help learners become better software engineers, not simply to assign grades.
