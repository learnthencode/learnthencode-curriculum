# Course Outline: Web Development

## Course Overview

Web Development is the second course in the LearnThenCode Full Stack Software Engineering Bootcamp.

This course introduces learners to the technologies that power the modern web.

Building upon the developer foundations established in the previous course, learners begin creating professional websites while developing a strong understanding of how browsers, web servers, and web standards work together.

The course focuses on building strong foundations in:

- Web fundamentals.
- HTML.
- Browser technologies.
- CSS.
- Responsive design.
- Accessibility.
- Professional frontend organization.

By the end of this course, learners should be able to build responsive, accessible, and well-organized websites using modern web standards.

---

# Course Philosophy

The purpose of this course is not simply to teach HTML and CSS.

Instead, learners develop an understanding of how websites are created, delivered, rendered, and maintained.

Every lesson contributes toward building and improving a single website, allowing learners to experience a realistic software development workflow while reinforcing previously learned concepts.

Learners should understand:

> Before building interactive applications, master the technologies that make the web possible.

---

# Learning Outcomes

By the end of this course, learners should be able to:

- Explain how the Internet and the World Wide Web work together.
- Describe how browsers communicate with web servers.
- Understand the request-response cycle.
- Create semantic HTML documents.
- Structure webpages using appropriate HTML elements.
- Use Browser Developer Tools to inspect and debug webpages.
- Style webpages using CSS.
- Build layouts using Flexbox and CSS Grid.
- Create responsive websites.
- Apply accessibility best practices.
- Organize frontend projects using professional standards.

---

# Course Structure

The course is organized into lessons.

Each lesson contains multiple sub-lessons and practical labs.

Rather than creating unrelated exercises, learners continuously improve the same website throughout the course.

Structure:

```
Web Development

│
├── Lesson
│   ├── Sub-lessons
```

# Lessons

## Lesson 1: Introduction to the Web

### Purpose

Help learners understand how the web works before building websites.

### Topics

- The Internet.
- The World Wide Web.
- Clients and servers.
- Browsers.
- URLs.
- Domain names.
- DNS.
- HTTP and HTTPS.
- The request-response cycle.

---

## Lesson 2: HTML Fundamentals

### Purpose

Introduce HTML as the language used to structure webpages.

### Topics

- HTML documents.
- Elements.
- Attributes.
- Headings.
- Paragraphs.
- Lists.
- Links.
- Images.

### Practical Skills

Learners should be able to create well-structured HTML webpages using semantic markup.

---

## Lesson 3: Browser Fundamentals and Developer Tools

### Purpose

Help learners understand how browsers interpret webpages and introduce the tools professional developers use to inspect and debug them.

### Topics

- Browser rendering.
- The Document Object Model (DOM).
- Browser Developer Tools.
- Inspecting elements.
- Debugging HTML.
- Browser compatibility.

### Practical Skills

Learners should be comfortable inspecting webpage structure and identifying HTML-related issues using Developer Tools.

---

## Lesson 4: Working with Content

### Purpose

Expand learners' ability to organize and present different types of information.

### Topics

- Tables.
- Forms.
- Multimedia.
- Embedded content.
- Semantic page structure.

### Practical Skills

Learners should be able to build richer webpages that effectively organize and present information.

---

## Lesson 5: CSS Fundamentals

### Purpose

Introduce CSS as the language used to control webpage presentation.

### Topics

- Selectors.
- Colors.
- Typography.
- Backgrounds.
- Borders.
- Spacing.
- Units.

### Practical Skills

Learners should be able to style webpages using reusable CSS rules.

---

## Lesson 6: CSS Layouts

### Purpose

Teach modern webpage layout techniques.

### Topics

- The Box Model.
- Display.
- Positioning.
- Flexbox.
- CSS Grid.
- Overflow.
- Z-index.

### Practical Skills

Learners should be able to create clean, responsive page layouts.

---

## Lesson 7: Responsive Web Design

### Purpose

Prepare learners to build websites that adapt to different screen sizes.

### Topics

- Mobile-first design.
- Responsive layouts.
- Viewports.
- Media queries.
- Relative units.
- Responsive typography.
- Responsive images.

### Practical Skills

Learners should be able to create websites that provide a consistent experience across desktops, tablets, and mobile devices.

---

## Lesson 8: Accessibility and Web Best Practices

### Purpose

Introduce professional standards for creating inclusive, maintainable, and high-quality websites.

### Topics

- Semantic HTML.
- Accessibility principles.
- Keyboard navigation.
- Alt text.
- Readable typography.
- Performance basics.
- SEO fundamentals.

### Practical Skills

Learners should be able to evaluate and improve the usability and accessibility of their websites.

---

## Lesson 9: Professional Frontend Project Organization

### Purpose

Teach learners how professional frontend projects are organized and maintained.

### Topics

- Project folder structure.
- Organizing assets.
- Naming conventions.
- CSS organization.
- Documentation.
- Maintainable code.

### Practical Skills

Learners should be able to organize frontend projects using professional development practices.

---

## Lesson 10: Building Your First Complete Website

### Purpose

Combine everything learned throughout the course into a complete website.

### Topics

- Planning website structure.
- Building multiple webpages.
- Applying responsive layouts.
- Improving accessibility.
- Refining visual design.
- Final testing.
- Project presentation.
