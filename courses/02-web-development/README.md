# Web Development

## Course Overview

Web Development is the second course in the LearnThenCode Full Stack Software Engineering Bootcamp.

After learning how professional software engineers work, learners now begin building software that people can use through the web.

This course introduces the technologies that form the foundation of every website:

- HTML
- CSS
- Responsive Web Design
- Browser fundamentals
- Web accessibility
- Professional frontend organization

Learners will explore how websites are structured, styled, rendered by browsers, and adapted to different devices.

Throughout the course, learners will continuously improve a website project, gradually transforming it from a simple HTML page into a professional, responsive website.

The goal is simple:

> **Understand how the web works before learning how to make it interactive.**

---

# Course Philosophy

Learning web development is about far more than memorizing HTML elements and CSS properties.

Professional web developers understand how browsers interpret web pages, why semantic markup matters, how users interact with websites, and how thoughtful design improves usability and accessibility.

This course focuses on helping learners understand the principles behind modern web development while developing practical skills through continuous hands-on practice.

Every lesson builds upon the previous one, allowing learners to progressively improve a single website as their knowledge grows.

---

By the end of this course, you will understand:
