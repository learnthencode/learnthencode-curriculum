# Lab: HTML Tables

## Overview

The LearnThenCode Café wants to display its drinks menu in a clean and organized way.

Instead of listing menu items as plain text, you will use an HTML table to present each drink alongside its price.

In this lab, you will practice creating a basic HTML table using rows and cells.

---

## Learning Objectives

By completing this lab, you will be able to:

- Create an HTML table.
- Add table rows using `<tr>`.
- Add table cells using `<td>`.
- Organize information into rows and columns.

---

## Scenario

You have been hired to improve the LearnThenCode Café website.

The café owner wants customers to easily view the available drinks and their prices.

Create a table that displays the following menu.

| Drink         |  Price  |
|---------------|---------|
| Tea           | KES 150 |
| Coffee        | KES 200 |
| Hot Chocolate | KES 250 |

---

## Instructions

Open the file:

```text
starter/index.html
```

Inside the `<body>`, create a table containing three rows.

Each row should contain:

- the drink name
- its price

Use only the following elements:

- `<table>`
- `<tr>`
- `<td>`

Do not use table headers (`<th>`) yet. You will learn about them in the next lab.

---

## Expected Output

Your webpage should display a simple table similar to:

| Tea | KES 150 |
|------|---------|
| Coffee | KES 200 |
| Hot Chocolate | KES 250 |

---

## Hints

- Every table begins with a `<table>` element.
- Each row is created using `<tr>`.
- Each piece of information goes inside a `<td>` element.

---

## Success Criteria

Before submitting your solution, make sure:

- [ ] The page contains one `<table>`.
- [ ] The table contains three rows.
- [ ] Each row contains two table cells.
- [ ] The table includes Tea.
- [ ] The table includes Coffee.
- [ ] The table includes Hot Chocolate.
