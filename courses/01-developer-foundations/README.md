# Developer Foundations

## Course Overview

Developer Foundations is the first course in the LearnThenCode Full Stack Software Engineering Bootcamp.

Before building applications, learners must first understand the environment, tools, and practices that professional software engineers use every day.

This course introduces the essential foundations required to begin a journey in software development, including software engineering concepts, development environments, command-line tools, version control, Git, and GitHub.

The goal is simple:

> **Understand how developers work before learning how developers build.**

---

# Course Philosophy

Software engineering is more than writing code.

Professional developers solve problems, collaborate with others, manage complex projects, and continuously improve their skills.

This course focuses on developing the mindset and habits required to become effective software engineers.

Learners will begin building the foundation that supports every technology they will learn throughout the bootcamp.

---

# What You Will Learn

By the end of this course, you will understand:

- What software is and how it is created.
- The difference between coding and programming.
- The difference between software development and software engineering.
- How computers, operating systems, and applications work together.
- How professional developers organize their work.
- How to use the command line.
- How version control helps developers manage changes.
- How to use Git for tracking code.
- How to use GitHub for collaboration.
- How to follow a professional development workflow.

# Course Topics

## Software Engineering Fundamentals

You will explore:

- What software engineers do.
- How software solves real-world problems.
- The software development lifecycle.
- The mindset of professional developers.

---

## Development Environment

You will learn how to prepare your computer for software development.

Topics include:

- Understanding development environments.
- Installing essential developer tools.
- Configuring your workspace.
- Verifying your setup.

Supported operating systems:

- Windows
- macOS
- Linux

---

## Command Line Fundamentals

You will learn how developers interact with computers through the terminal.

Topics include:

- Navigating files and folders.
- Creating and managing files.
- Running commands.
- Understanding shells.

---

## Version Control with Git

You will learn how developers track and manage changes in their projects.

Topics include:

- Repositories.
- Commits.
- History.
- Branches.
- Managing changes.

---

## Collaboration with GitHub

You will learn how developers share and collaborate on software projects.

Topics include:

- Remote repositories.
- Pushing and pulling code.
- Cloning projects.
- Issues.
- Pull requests.
- Open-source workflows.
