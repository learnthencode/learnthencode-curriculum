# Developer Profile Repository

## Overview

Throughout Developer Foundations, you have learned the foundations required to begin working like a professional developer.

You have learned:

- How software engineers think.
- How computers and software work.
- How developers organize projects.
- How to use the command line.
- How to use Git.
- How to collaborate using GitHub.

In this activity, you will create your first professional developer profile repository.

This repository will introduce you to the developer community and document your learning journey.

---

# Objective

By completing this activity, you will practice:

- Creating a GitHub repository.
- Organizing project files.
- Writing documentation using Markdown.
- Using Git to track changes.
- Sharing your work professionally.

---

# Project Requirements

## Create a GitHub repository named:

```text
developer-profile
```

Your repository should contain:

```text
developer-profile
├── README.md
├── learning-journey.md
└── goals.md
```

---

# README.md

Your README should introduce you as a developer.

Include:

## Introduction

Write a short introduction about yourself.

Example:

- Your name.
- Your interest in software development.
- Why you started learning programming.

---

## Technical Interests

Explain areas of technology you are interested in.

Examples:

- Web development.
- Mobile development.
- Artificial intelligence.
- Software engineering.

---

## Current Learning Journey

Explain what you are currently learning.

Mention:

- Courses completed.
- Skills being developed.
- Technologies being explored.

---

# learning-journey.md

Document your experience learning software development.

Include:

- Important concepts you learned.
- Challenges you faced.
- Lessons you discovered.
- Skills you improved.

---

# goals.md

Describe your future goals as a developer.

Include:

- Technologies you want to learn.
- Projects you want to build.
- The type of developer you want to become.

# Git Workflow

Use the workflow learned throughout this course:

## Create Repository

Create the repository on GitHub.

---

## Clone Repository

Clone it to your computer.

```bash
git clone repository-url
```

## Add Files

Create your documentation files.


## Track Changes

Check your changes:

```bash
git status
```

## Stage your files:

```bash
git add .
```

## Commit Changes

Create a meaningful commit:

```bash
git commit -m "Create developer profile documentation"
```

## Push Changes

Upload your work:
```bash
git push
```
